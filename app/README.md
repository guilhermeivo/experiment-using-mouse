## NPM Install

```bash
$ npm i -D node-sass
$ npm i -D webpack webpack-cli
$ npm i -D css-loader style-loader sass-loader
$ npm i -D file-loader
# OR
$ npm i -D
```

## NPM Run
```bash
$ npm run builds
```
