## NPM Install
```bash
$ npm i dotenv

$ npm i -D node-sass
$ npm i -D webpack webpack-cli
$ npm i -D css-loader style-loader sass-loader
$ npm i -D file-loader
# OR
$ npm i
$ npm i -D
```

## NPM Run
```bash
$ npm run builds
```

## Environment configuration
`.env`
```
NODE_ENV="DEVELOPMENT"
# OR
NODE_ENV="PRODUCTION"
```
